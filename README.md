# GsmRanking

# 제작동기

깃허브 관리를 하지 않는 친구들이 자신의 친구,선배,후배의 랭킹을 보면서 나도 관리를 하여 저 랭킹에 나의 이름을 올려보자 라는 동기를 얻을수 있으면 좋을거 같아 제작하였습니다.

# 완성 기능

* 랭킹 10위 출력
* 자신의 git 랭킹 등록
* 프로필 클릭시 프로필 주인의 깃헙 페이지로 넘어가기

# 사용스킬
* MVVM
* Apollo(Graphql)
* Hilt
* coroutine
* okHttp


학교 학생들의 커밋수를 측정해서 랭킹을 나타내는 앱입니다.
